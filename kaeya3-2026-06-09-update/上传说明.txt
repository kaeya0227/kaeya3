今天要上传到 kaeya0227/kaeya3 的是这个 data 文件夹。

最简单做法：
1. 打开 https://github.com/kaeya0227/kaeya3
2. 点 Add file -> Upload files
3. 把本文件夹里的 data 文件夹整个拖进去
4. Commit message 可以写：Update literature radar for 2026-06-09
5. 点 Commit changes

会更新/新增的文件：
- data/latest.json
- data/runs.json
- data/archive/2026-06-09.json

这些路径和原网页使用的数据路径一致。latest.json 和 runs.json 会覆盖旧版本，archive/2026-06-09.json 是今天新增的历史归档。
